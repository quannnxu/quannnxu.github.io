Hive

order by:全局排序

sort by:分区排序,整体无序.

distribute by:按照规则进行分组,通常和sort by连用

cluster by:等于distribute by a order by a(同一字段),不能使用降序

严格模式:

set hive.mapred.mode = strict;

严格模式开启下禁用order by

设置reduce个数:

set mapred.reduce.tasks=3;

把数据写到本地:

insert overwrite local directory ‘path’ row format delimited fields terminated by ‘\t’ select * from xx;

nohup sh hiveserver2 &

./beeline -u jdbc:hive2://warehhouse001:10000/emp -n hadoop

如果想开启就能使用必须编译源码,把自定义方法配置到FunctionRegistry.java中

也可以编写初始化脚本hive-init.sql在hive执行时运行:

add jar /home/hadoop/lib/SayHell-1.0.jar;
CREATE TEMPORARY FUNCTION say_hello3 AS "com.example.hadoop.hdfs.UDF.sayHello";

启动时执行:

hive -i hive-init.sql

这样每次启动后就可以使用,这是一种偷懒的方法.
