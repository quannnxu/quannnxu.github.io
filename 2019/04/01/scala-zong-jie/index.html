二 Classs VS Object

在Scala中没有静态修饰符,在object下的成员全部都是静态的,如果在类中声明了与该类相同名字的object则该object是该类的”伴生对象”，所以对于一个class来说，所有的方法和成员变量在实例被 new 出来之前都是无法访问的.

可以理解为Scala把类中的static集中放到了object对象中,伴生对象和类文件必须是同一个源文件,可以用伴生对象做一些初始化操作.

object不能提供构造器参数,也就是说object必须是无参的

在Scala中,类名可以和对象名为同一个名字,该对象称为该类的伴生对象,类和伴生对象可以相互访问他们的私有属性,但是它们必须在同一个源文件中.

使用的区别:

    //这里调用的是class
    val  a = new ApplyTest
    println(a)
    //这里调用的是object
    val b = ApplyTest
    println(b)

Object不用new就可以实例化,而Class需要new

 // new 对象() => 调用的是class apply方法
    val  e = new ApplyTest()
    e()    //调用的是class apply方法
 // 对象() => 调用的是class apply方法
    val  f =ApplyTest()
    //这里是调用的class apply方法
    val array = new Array[String](5)
    //这里是调用的object apply方法
    val array2 = Array(1,2,3)

这里object好处就体现出来了,不用new就可以直接实例化

/**
 * object修饰的东西,调用时不需要new,直接Timer.属性/方法
 */
object Timer&#123;
  var cnt = 0

  def currentCnt=&#123;
    cnt += 1
    cnt
  &#125;
&#125;

总结:

1.object不需要new就可以实例化,class必须new才可以实例化.

2.object必须是无参构造器,而class可以有参也可以无参.

3.object修饰的可以不用实例化,直接类名.属性/方法调用,class必须实例化才可以调用.

七 Option、Some、None

Option是为了避免NullPointerException

Some是一个继承Option的case class

None是一个继承了Option[Nothing]的case object

Option和Some的用法:

val a :Option[String] = Some("Some")

    println(a.get)

定义了一个类型为String的Option，然后用Some给它赋了一个值。使用Option的get方法来获取这个值.

Option和None的用法:

    val b : Option[String] = None
    println(b.getOrElse("None"))

定义了一个Option，并给给他赋值None。在获取值的时候，我们没有调用get方法，相反我们使用的是getOrElse，如果值为空，则给他一个默认值。None没有get方法,不能直接调用.

在Scala里Option[T]实际上是一个容器，就像数组或是List一样，你可以把他看成是一个可能有零到一个元素的List。
当你的Option里面有东西的时候，这个List的长度是1（也就是 Some），而当你的Option里没有东西的时候，它的长度是0（也就是 None）

4.PartialFunction

偏函数

1）在对符合某个条件，而不是所有情况进行逻辑操作时，使用偏函数是一个不错的选择

2）将包在大括号内的一组case语句封装为函数，我们称之为偏函数，它只对会作用于指定类型的参数或指定范围值的参数实施计算，超出范围的值会忽略（未必会忽略，这取决于你打算怎样处理）

3）偏函数在Scala中是一个特质PartialFunction

val array:Array[Any] = Array(1,2,"X",10.0)

    val partialFunction=new PartialFunction[Any,Int] &#123;
      override def isDefinedAt(x: Any): Boolean = &#123;
        if (x.isInstanceOf[Int]) true else false
      &#125;

      override def apply(v1: Any): Int = v1.asInstanceOf[Int] * 5
    &#125;

    array.collect(partialFunction).foreach(println)

    //collect调用偏函数
    array.collect(&#123;
      case i:Int => i * 5
    &#125;).foreach(println)
